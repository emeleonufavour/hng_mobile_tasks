## Setup

- Clone the repository

  ```bash
  git clone https://github.com/emeleonufavour/hng_mobile_tasks.git
  cd hng-mobile-app
  ```

- Install dependencies

  ```bash
  flutter pub get
  ```

- Run the app

  ```bash
  flutter run
  ```
